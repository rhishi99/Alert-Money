# /agy:delegate — Delegate a coding task to Antigravity

You are delegating a task to Google Antigravity CLI (agy).

Use the bash tool to run:

```bash
agy -p "You are working in the NxBagger project at e:/vibe-code-projects/NxBagger. Task: $ARGUMENTS"
```

Wait for the response and summarize the result for the user.
If agy output is empty, wrap with winpty: `winpty agy -p "..."`.
